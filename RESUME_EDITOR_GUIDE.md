# 简历编辑器使用指南

## 概述

简历编辑系统已经升级为更细化、更专业的编辑界面，支持以下几种简历板块类型：

## 支持的简历板块类型

### 1. 个人信息 (Profile)
**包含字段：**
- 头像上传
- 姓名
- 职位/头衔
- 个人简介
- 所在地
- 邮箱
- 电话
- 个人网站

**特点：**
- 支持图片上传功能
- 完整的联系方式管理
- 适合展示个人基本信息

### 2. 工作经历 (Work Experience)
**包含字段：**
- 公司名称
- 职位
- 开始日期
- 结束日期（支持"目前在职"选项）
- 工作描述

**特点：**
- 支持添加多条工作经历
- 时间线展示
- 可以标记当前在职状态

### 3. 教育背景 (Education)
**包含字段：**
- 学校名称
- 学位（本科/硕士/博士）
- 专业
- 开始日期
- 结束日期（支持"在读"选项）
- GPA/成绩
- 补充说明

**特点：**
- 支持添加多条教育经历
- 可以添加荣誉和项目说明
- 支持在读状态标记

### 4. 技术栈 (Tech Stack)
**包含字段：**
- 技术分类（如：前端开发、设计工具、编程语言）
- 技术列表
  - 技术名称
  - 掌握程度（入门/熟练/精通/专家）

**特点：**
- 按分类组织技术
- 每个技术可标记掌握程度
- 支持多个分类和多项技术

### 5. 专业技能 (Skills)
**包含字段：**
- 技能名称
- 技能描述

**特点：**
- 适合描述软技能和核心能力
- 支持添加多项技能
- 可以详细描述每项技能

## 使用方法

### 添加新的简历板块

1. 点击"添加内容"按钮
2. 选择要添加的板块类型（个人信息、工作经历、教育背景、技术栈、专业技能）
3. 填写相应的表单内容
4. 点击"创建"保存

### 编辑现有板块

1. 在简历列表中找到要编辑的板块
2. 点击"编辑"按钮
3. 修改内容
4. 点击"更新"保存

### 删除板块

1. 在简历列表中找到要删除的板块
2. 点击"删除"按钮
3. 确认删除操作

## 界面特点

### 分类展示
- 简历内容按类型分组显示
- 每个类型有对应的图标标识
- 显示每个类型下的板块数量

### 可视化内容
- **个人信息**：显示头像、姓名、职位和联系方式
- **工作经历**：时间线样式展示，包含公司、职位和时间
- **教育背景**：显示学校、学位、专业和时间
- **技术栈**：标签云样式展示技术和掌握程度
- **专业技能**：列表展示技能名称和描述

### 响应式设计
- 适配不同屏幕尺寸
- 移动端友好的表单布局
- 清晰的视觉层次

## 数据结构

所有简历数据存储在 `resume_sections` 表中：

```sql
CREATE TABLE resume_sections (
    id UUID PRIMARY KEY,
    section_type VARCHAR(50) NOT NULL,  -- profile, work_experience, education, tech_stack, skills
    title VARCHAR(255),                  -- 板块标题
    content TEXT,                        -- JSON字符串格式的内容
    metadata JSONB,                      -- 结构化的数据
    display_order INTEGER DEFAULT 0,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);
```

## 技术实现

### 组件结构

```
components/
├── ResumeForm.tsx              # 主表单组件，包含类型选择器
└── resume/
    ├── ProfileForm.tsx         # 个人信息表单
    ├── WorkExperienceForm.tsx  # 工作经历表单
    ├── EducationForm.tsx       # 教育背景表单
    ├── TechStackForm.tsx       # 技术栈表单
    └── SkillsForm.tsx          # 专业技能表单
```

### 表单验证

使用 `zod` 进行表单验证，确保数据完整性：
- 必填字段验证
- 邮箱格式验证
- URL格式验证
- 日期格式验证

### 图片上传

个人信息板块支持头像上传：
- 支持 JPG、PNG、WebP 格式
- 最大文件大小：10MB
- 自动上传到 Supabase Storage
- 实时预览功能

## 最佳实践

1. **完整填写个人信息**：确保联系方式准确，方便他人联系
2. **详细描述工作经历**：突出关键成就和职责
3. **合理分类技术栈**：按照前端、后端、工具等分类组织
4. **量化技能水平**：使用掌握程度标记，让技能评估更客观
5. **定期更新**：保持简历内容的时效性

## 注意事项

- 图片上传需要网络连接
- 删除操作不可恢复，请谨慎操作
- 建议定期备份重要数据
- 表单中的必填字段必须填写才能保存
