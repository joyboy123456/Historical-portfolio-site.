## 1. 设计方向与理由

### 1.1 风格精髓

**Editorial（杂志编辑风格极简）** 以排版为核心，通过大胆的字体层级对比、精致的行距字距和适度的衬线字体运用，为作品集营造出专业、优雅、易读的阅读体验。留白不是简单的空白，而是服务于内容叙事节奏的视觉呼吸。黑白灰的克制配色让作品图片成为色彩的主角，单一点缀色仅用于关键交互，体现编辑级别的视觉纪律。

**参考案例：** Medium.com（优雅排版）、Kinfolk杂志官网（温和极简）、The Great Discontent（编辑质感）

### 1.2 设计理念

1. **排版主导：** 衬线字体建立文学气质，字号对比（64-96px标题 vs 18-20px正文）创造戏剧性层级
2. **阅读节奏：** 1.6-1.8行高、24-32px段落间距，确保长时间阅读的舒适性
3. **图文平衡：** 高质量作品图片 + 精心编排的文字说明，避免视觉噪声
4. **细节雕琢：** 斜体强调、细线分隔、轻微阴影，体现专业克制

## 2. 设计令牌

### 2.1 色彩系统

**分布策略：** 85% 中性色 + 10% 温和背景 + 5% 点缀色

| 色彩类型 | 色值 | 用途 | WCAG对比度 |
|---------|------|------|-----------|
| **文本色** |||
| text-primary | `#1A1A1A` | 正文、标题 | 15.8:1 (AAA) |
| text-secondary | `#4A4A4A` | 副标题、元数据 | 8.2:1 (AAA) |
| text-tertiary | `#6B6B6B` | 图片说明、辅助文字 | 5.5:1 (AA+) |
| **背景色** |||
| bg-primary | `#FEFEFE` | 页面主背景（温暖白） | - |
| bg-surface | `#F9F9F7` | 卡片背景（奶油色调） | - |
| bg-divider | `#E5E5E0` | 分隔线、边框 | - |
| **点缀色（单一强调色）** |||
| accent-primary | `#B8001F` | CTA按钮、活跃链接 | 6.1:1 (AA) |
| accent-hover | `#8F0018` | 按钮悬停状态 | - |

**色彩规则：**
- ❌ 禁止：多个点缀色、渐变背景、霓虹色
- ✅ 允许：作品图片全彩、单一强调色

### 2.2 字体系统

**字体家族：**

| 类型 | 字体栈 | 字重 | 用途 |
|------|--------|------|------|
| 标题衬线 | `'Playfair Display', Georgia, serif` | 400, 600, 700 | H1-H3、引用 |
| 正文衬线 | `Georgia, Charter, serif` | 400, 400 italic | 正文、说明文字 |
| UI无衬线 | `'Inter', -apple-system, sans-serif` | 400, 600 | 导航、按钮、元数据 |

**字号层级（桌面端 1920px）：**

| 层级 | 字号 | 字重 | 行高 | 字间距 | 用途 |
|------|------|------|------|--------|------|
| Display (h1) | 64-80px | Playfair Bold 700 | 1.1 | -0.01em | Hero区标题 |
| Headline (h2) | 40-48px | Playfair Semibold 600 | 1.2 | 0 | 区域标题 |
| Subhead (h3) | 24-28px | Playfair Regular 400 | 1.3 | 0 | 卡片标题 |
| Body Large | 21px | Georgia Regular 400 | 1.7 | 0 | 引导段落 |
| Body | 18px | Georgia Regular 400 | 1.6 | 0 | 正文、卡片描述 |
| Small | 16px | Georgia Regular 400 | 1.6 | 0 | 图片说明 |
| Metadata | 14px | Inter Regular 400 | 1.5 | 0.01em | 日期、标签 |

**移动端调整（≤768px）：**
- Display: 40-48px
- Headline: 32-36px
- Body: 18px（保持可读性）

### 2.3 间距系统（8pt网格）

| Token | 值 | 用途 |
|-------|-----|------|
| spacing-xs | 8px | 行内元素间距 |
| spacing-sm | 16px | 紧密组件间距 |
| spacing-md | 24px | 段落间距 |
| spacing-lg | 32px | 引用、图片边距 |
| spacing-xl | 48px | 小区域分隔 |
| spacing-2xl | 64px | 主要区域分隔 |
| spacing-3xl | 96px | Hero与作品展示区间距 |

**阅读宽度：**
- 文本列最大宽度：650-750px（60-75字符/行）
- 页面侧边距：10-15%视口宽度

### 2.4 其他令牌

**圆角（Editorial 保持锐利）：**
- radius-sm: 4px（按钮、卡片）
- radius-md: 8px（大卡片）
- radius-full: 50%（圆形头像）

**阴影（轻微克制）：**
- shadow-card: `0 4px 12px rgba(0, 0, 0, 0.08)`
- shadow-card-hover: `0 8px 24px rgba(0, 0, 0, 0.12)`

**边框：**
- border-thin: `1px solid #E5E5E0`

## 3. 组件规范

### 3.1 导航栏（Navigation Bar）

**结构：** 固定顶部导航 + Logo左对齐 + 右侧菜单链接

**尺寸与间距：**
- 高度：64-72px
- Logo高度：32px
- 链接间距：32-40px

**样式：**
- 背景：`#FEFEFE`
- 底部边框：`1px solid #E5E5E0`
- Logo：Inter Semibold 600, 20px 或图片
- 导航链接：Inter Regular 400, 16px, `#1A1A1A`
- 活跃链接：`#B8001F` 下划线

**状态：**
- 默认：黑色文字
- 悬停：出现 `#B8001F` 下划线（2px，过渡200ms）
- 活跃：红色文字 + 下划线

**移动端（≤768px）：**
- 切换为汉堡菜单
- 菜单展开：全屏覆盖层，垂直堆叠链接

### 3.2 Hero区域（首屏）

**布局模式：** 居中对齐 + 圆形头像 + 标题 + 副标题 + CTA

**尺寸：**
- 区域高度：70-80vh（最小500px）
- 头像直径：120-160px
- 标题：64-80px Playfair Bold
- 副标题：21px Georgia Regular
- CTA按钮：见§3.5

**间距：**
- 头像 → 标题：32px
- 标题 → 副标题：16px
- 副标题 → CTA：32px

**样式：**
- 背景：`#FEFEFE` 或 `#F9F9F7`
- 头像：圆形裁剪（border-radius: 50%），细边框 `1px solid #E5E5E0`
- 标题：深黑色（`#1A1A1A`），行高1.1
- 副标题：次要文本色（`#4A4A4A`），斜体可选

### 3.3 作品卡片（Portfolio Card）

**布局：** 3列网格（桌面）→ 2列（平板）→ 1列（移动）

**卡片结构：**
- 图片（顶部，3:2或16:9比例）
- 标题（24-28px Playfair）
- 描述（16-18px Georgia，2-3行截断）
- 标签（14px Inter，浅灰背景）

**尺寸：**
- 内边距：24-32px
- 图片圆角：4px
- 卡片圆角：8px

**样式：**
- 背景：`#F9F9F7`（奶油色）
- 边框：`1px solid #E5E5E0`
- 阴影：`0 4px 12px rgba(0, 0, 0, 0.08)`

**悬停状态（桌面）：**
- 向上移动：`translateY(-4px)`
- 阴影增强：`0 8px 24px rgba(0, 0, 0, 0.12)`
- 过渡：`all 300ms cubic-bezier(0.4, 0.0, 0.2, 1)`

**移动端：**
- 无悬停效果
- 保持静态阴影

### 3.4 区域标题（Section Headline）

**样式：**
- 字体：Playfair Display Semibold 600
- 字号：40-48px（桌面），32-36px（移动）
- 行高：1.2
- 颜色：`#1A1A1A`
- 下方间距：48-64px

**装饰（可选）：**
- 底部细线分隔：`3px solid #B8001F`，居中，宽度80-100px
- 与标题间距：16px

### 3.5 按钮（CTA Button）

**主要按钮（Primary）：**
- 高度：48px
- 水平内边距：24-32px
- 圆角：4px
- 字体：Inter Semibold 600, 16px, 字间距 0.02em
- 文字：全大写可选（如 "VIEW WORK"）
- 背景：`#B8001F`
- 文字色：`#FFFFFF`
- 无边框

**悬停状态：**
- 背景：`#8F0018`（深化15%）
- 轻微缩放：`scale(1.02)`
- 过渡：200ms ease-out

**次要按钮（Outline）：**
- 同样尺寸
- 背景：透明
- 边框：`2px solid #1A1A1A`
- 文字色：`#1A1A1A`
- 悬停：反转（黑底白字）

**触控目标：**
- 最小 44×44px（移动端）
- 间距：16px（多按钮并列时）

### 3.6 页脚（Footer）

**布局：** 居中对齐 + 社交图标 + 版权文字

**尺寸：**
- 高度：120-160px
- 顶部边框：`1px solid #E5E5E0`

**样式：**
- 背景：`#FEFEFE`
- 社交图标：SVG，20-24px，`#4A4A4A`
- 图标间距：24px
- 版权文字：Inter Regular 400, 14px, `#6B6B6B`

**图标悬停：**
- 颜色变为 `#B8001F`
- 过渡：200ms

## 4. 布局与响应式

### 4.1 单页滚动结构

**区域顺序：**
1. Navigation（固定顶部）
2. Hero Section（70-80vh）
3. Portfolio Grid Section（自适应高度）
4. About Section（可选，50-60vh）
5. Contact Section（50vh）
6. Footer（120-160px）

**区域间距：**
- Hero → Portfolio：96px
- Portfolio → About：64px
- About → Contact：64px
- Contact → Footer：48px

### 4.2 网格系统

**作品展示网格（Portfolio Grid）：**

**桌面（≥1024px）：**
- 3列布局
- 列间距（gap）：32px
- 行间距：48px
- 容器最大宽度：1200px，居中

**平板（768-1023px）：**
- 2列布局
- 列间距：24px
- 行间距：40px

**移动（≤767px）：**
- 1列布局
- 卡片宽度：100%（左右边距16px）
- 行间距：32px

### 4.3 断点策略

```
sm:  640px  （小手机 → 1列）
md:  768px  （平板 → 2列）
lg:  1024px （桌面 → 3列）
xl:  1280px （大屏 → 维持3列，增加间距）
```

### 4.4 响应式规则

**移动优先（Mobile-First）：**
- 基础样式为移动端（单列、18px正文、44px触控目标）
- 使用 `min-width` 媒体查询逐步增强

**容器宽度：**
- 移动：100%（边距16px）
- 平板：90%（最大720px）
- 桌面：85%（最大1200px）

**导航适配：**
- 桌面：水平链接
- 移动：汉堡菜单 + 全屏抽屉

**Hero头像：**
- 桌面：160px
- 移动：120px

## 5. 交互与动效

### 5.1 动画时长

| 速度 | 时长 | 用途 |
|------|------|------|
| Fast | 200ms | 按钮悬停、链接下划线 |
| Standard | 300ms | 卡片悬停、阴影变化 |
| Slow | 400ms | 页面过渡、菜单展开 |

### 5.2 缓动函数

**首选：** `ease-out`（快进慢出，适合大多数交互）  
**精致：** `cubic-bezier(0.4, 0.0, 0.2, 1)`（Material Design标准）

### 5.3 允许的动效

**✅ 推荐：**
- 卡片悬停上浮（-4px）+ 阴影增强
- 按钮悬停颜色深化 + 轻微缩放
- 导航链接下划线滑入（从左到右）
- 滚动时区域淡入（opacity 0→1，移动端可选）
- 平滑锚点滚动（scroll-behavior: smooth）

**❌ 禁止：**
- 过度动画（旋转、弹跳、炫技效果）
- 影响阅读的视差滚动
- 自动播放的轮播图
- 加载时的复杂动画

### 5.4 触控反馈

**移动端：**
- 点击按钮：轻微缩放（scale 0.98）
- 卡片点击：无悬停效果，直接跳转
- 导航链接：触控区域扩大至48×48px

### 5.5 无障碍支持

- 支持 `prefers-reduced-motion`（禁用装饰性动画）
- 键盘焦点可见（outline: 2px solid `#B8001F`）
- 焦点顺序：Logo → 导航 → Hero CTA → 作品卡片 → 联系 → 社交图标

## 6. 设计检查清单

**排版：**
- ✅ 衬线标题（Playfair）+ 衬线正文（Georgia）
- ✅ 字号对比 ≥2.5倍（80px标题 vs 18px正文）
- ✅ 正文行高 1.6-1.8
- ✅ 段落间距 24-32px
- ✅ 文本列宽 ≤750px

**色彩：**
- ✅ 85% 黑白灰 + 10% 温和背景 + 5% 点缀色
- ✅ 点缀色仅用于CTA和活跃链接
- ✅ 所有文本对比度 ≥4.5:1（正文 ≥7:1）

**间距：**
- ✅ 8pt网格（8, 16, 24, 32, 48, 64, 96）
- ✅ 区域间距 64-96px
- ✅ 卡片内边距 24-32px

**组件：**
- ✅ 导航高度 64-72px
- ✅ 圆形头像 120-160px
- ✅ 卡片圆角 8px，轻微阴影
- ✅ 按钮高度 48px，圆角 4px
- ✅ 触控目标 ≥44px

**响应式：**
- ✅ 3列（桌面）→ 2列（平板）→ 1列（移动）
- ✅ 移动端正文 ≥18px
- ✅ 汉堡菜单（移动端）

**动效：**
- ✅ 卡片悬停上浮 -4px + 阴影增强
- ✅ 过渡时长 200-300ms
- ✅ 缓动 ease-out
- ✅ 支持 reduced-motion
