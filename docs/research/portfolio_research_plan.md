# 极简主义设计师作品集网站案例研究计划

## 研究目标
收集和分析优秀的极简主义设计师作品集网站案例，重点关注：
1. 知名设计师的个人作品集网站
2. 电商设计师的作品集网站案例  
3. 极简风格的创意作品集设计
4. 响应式设计和移动端适配方案
5. 作品展示的创新布局方式
